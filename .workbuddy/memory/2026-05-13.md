# 2026-05-13 — LLM News Tracker MVP Built

## What was done
- Built complete LLM News Tracker from scratch using the before-vc workflow
- Implemented 9 YouTube channel monitoring via RSS + yt-dlp subtitle download
- Integrated Gemini 2.5 Flash AI for video content analysis (topic classification, Chinese summaries, speaker ID)
- Created clean static HTML table with search/filter/topic-filter/channel-filter
- Deployed to GitHub Pages: https://barrychanzzz.github.io/llm-news-tracker/
- Set up GitHub Actions CI/CD workflow (every 12 hours) — pending manual workflow file push due to OAuth scope limitation
- 39 videos fetched with subtitles, 22 analyzed by Gemini (17 pending due to free tier daily limit of 20 requests)

## Key decisions
- Used `requests` library instead of feedparser's built-in HTTP to fix SSL issues on macOS
- Switched from `google-generativeai` (deprecated) to `google-genai` SDK
- Used `gemini-2.5-flash` model (1.5-flash no longer available)
- Renamed `site/` to `docs/` for GitHub Pages compatibility (branch deployment only supports `/` or `/docs`)

## Remaining
- User needs to manually add `.github/workflows/update.yml` to the repo to enable auto-updates
- Gemini free tier daily limit of 20 requests — remaining 17 videos will be analyzed in subsequent runs
